Audio Mode

  • MIDI Out: MIDI output only (select destination below)
  • Both: Internal audio + MIDI output simultaneously

Envelope Controls

Attack
Controls how quickly notes fade in (0.001s - 1.0s). Lower values = sharper attack.
Release
(Only visible when Auto Strum is enabled)
Controls how quickly notes fade out (0.0s - 1.0s). Higher values = longer sustain.
Strum Speed
(Only visible when Auto Strum is enabled)
Controls the speed of automatic arpeggiation (0% = instant chord, 100% = slowest).

Toggle Buttons

Four buttons that glow orange when active:

HOLD

Enabled by default. When on, chords stay active until you tap another chord. When off, chords release when you lift your finger.

STRUM

Enable automatic chord arpeggiation. Direction picker appears above when enabled. Disables manual strumplate.

FILTER

Enabled by default. Applies a low-pass filter to smooth the audio output.

FX

Disabled by default. Enables reverb/delay effects processing.

Direction Picker

(Only visible when Auto Strum is enabled)

  • Forward: Lowest to highest note, plays once
  • Backward: Highest to lowest note, plays once
  • Ping-Pong: Continuous back-and-forth looping

Voicing (Inversions)

Four buttons (I, II, III, IV) that glow orange when selected. Changes which note of the chord is the lowest:

  • I: Root position (default)
  • II: First inversion
  • III: Second inversion
  • IV: Third inversion (when available)

Not all chords have all inversions available. Unavailable inversions are hidden.

MIDI Out

(Only visible when Audio Mode is set to "MIDI Out" or "Both")

Select your MIDI destination device from the dropdown. Shows "No devices available" when no MIDI destinations are found.

Chord Grid

The chord grid is the heart of Chordea. Each button represents one chord voicing.

Layout

  • Rows: Chord qualities (Major, Minor, Dominant7, etc.)
  • Columns: Root notes (C, C‚ôØ, D, D‚ôØ, E, F, F‚ôØ, G, G‚ôØ, A, A‚ôØ, B)

Playing Chords

  • Tap: Select and play a chord
  • Drag: Smoothly transition between chords (no repeated triggers on same chord)
  • Hold Mode ON: Chord stays selected until you tap another
  • Hold Mode OFF: Chord releases when you lift your finger

Visual States

  • Cream/Grey buttons: White keys / Black keys (natural vs sharp)
  • Yellow glow: Currently selected chord
  • Yellow outline: Suggested next chord (based on music theory)
  • Green: Chord currently being learned/recorded

Suggestion Mode

When suggestion filter is enabled (+), the grid transforms:

  • Grid reorganizes into vertical columns by note
  • Only shows the selected chord + suggested next chords
  • Chords appear larger (double height) for easier selection
  • Perfect for focused, theory-guided playing
üí° Pro Tip: Use suggestion mode with Auto Strum disabled and Hold disabled for rapid-fire chord changes. The last played chord stays visible (without glow) so you can quickly re-trigger it.

Strumplate

The vertical strumplate on the right side provides expressive, multi-touch note triggering.

How It Works

  • Touch anywhere: Triggers notes based on vertical position
  • Bottom = low notes, Top = high notes
  • Multi-touch enabled: Touch multiple positions simultaneously for polyphonic playing
  • Refractory period: 50ms per note prevents unwanted re-triggers

Note Mapping

The strumplate intelligently maps your touch position to unique chord notes across 4 octaves. For a C Major chord:

  • Bottom region: C2, E2, G2
  • Middle-low: C3, E3, G3
  • Middle-high: C4, E4, G4
  • Top region: C5, E5, G5

The surface subdivides dynamically based on the number of unique notes available.

Auto Strum Mode

When Auto Strum is enabled, the strumplate shows "AUTO STRUM ON" and is disabled. The app automatically triggers notes based on your chosen direction and speed.

üí° Pro Tip: For expressive playing, try sliding your finger vertically while touching the strumplate to create harp-like runs through the chord.

Learn Mode & Sequences

Learn Mode allows you to record and save up to 24 chord sequences (banks) for instant recall.

Recording a Sequence

  1. Press the L button (turns red)
  2. Play up to 16 chords by tapping them in the chord grid
  3. Chords turn green as they're recorded
  4. Duplicate consecutive chords are automatically skipped
  5. Press L again to stop recording, or it auto-stops at 16 chords

Viewing Sequences

After recording (or press S button), the learned chord view appears:

  • Chords displayed in a dynamic grid layout (1-16 chords)
  • Each chord shows its position number (1-16)
  • Swipe left/right to switch between sequences
  • Tap any chord to play it
  • Drag across chords for smooth transitions
  • Press ‚úï button (orange) to exit back to normal view

Sequence Deck

At the bottom of learned view, see all 24 sequence slots:

  • Numbered 1-24
  • Grey filled: Sequence contains chords (alternating shades for visibility)
  • Dotted outline: Empty slot
  • Yellow border: Currently active sequence
  • Tap: Switch to that sequence
  • Long press: Overwrite that sequence (enters learn mode for that slot)

Managing Sequences

  • All sequences are saved automatically and persist across app sessions
  • Switching to an empty slot automatically starts learn mode for that slot
  • If all 24 banks are full, a warning appears: "all banks full, clear a bank to learn"
  • To clear a bank: Long press the sequence number and record a new sequence (or exit learn mode immediately to leave it empty)
üí° Pro Tip: Use sequences to organize chord progressions by song, key, or mood. The 24 banks give you plenty of room for a full setlist!

Sound Design Tips

Troubleshooting

No Sound

  • Check iPad volume and mute switch
  • Verify Audio Mode is set to "Audio" or "Both" (not "MIDI Out")
  • Check Volume slider in control panel
  • Make sure a chord is selected (yellow glow)
  • Touch the strumplate (if Auto Strum is off)

MIDI Not Working

  • Verify MIDI device is connected and powered on
  • Set Audio Mode to "MIDI Out" or "Both"
  • Select correct MIDI destination from dropdown
  • Try disconnecting and reconnecting MIDI device
  • Restart Chordea if destination doesn't appear

Strumplate Not Responding

  • Check if Auto Strum is enabled (disables manual strumplate)
  • Verify a chord is selected
  • Make sure you're touching within the strumplate area (right 10% of screen)

Chord Grid Disappeared

  • Check Chord Group Filter - at least one group (Core/Color/Tension) must be selected
  • If using Suggestion Filter (+), make sure a chord is selected or has been played

Performance Issues

  • Close other apps to free up iPad resources
  • Disable FX if experiencing audio glitches
  • Reduce voice count by using simpler chord qualities
  • Restart the app if performance degrades over time
